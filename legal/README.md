# Legal de VYRA

Este directorio contiene la fuente legal activa que usa el producto:

- `PRIVACY_POLICY.md`
- `TERMS_OF_SERVICE.md`
- `medical_disclaimers.json`

## Uso actual

- Las pantallas legales visibles de la app leen `PRIVACY_POLICY.md` y `TERMS_OF_SERVICE.md`.
- Los avisos modulares y de primer uso leen `medical_disclaimers.json`.
- El naming canónico actual para la capa de IA es `context_ai`.

## Criterios de mantenimiento

- No dejar ejemplos históricos, snippets obsoletos ni instrucciones de despliegue mezcladas con la fuente legal.
- Si cambia una URL pública o un flujo de aceptación, actualizar la pantalla correspondiente y este README el mismo día.
- Si se agrega un nuevo disclaimer, debe quedar identificado por clave estable y con `shown_in` explícito.

## Pendiente externo

- Hosting público definitivo de `PRIVACY_POLICY.md` y `TERMS_OF_SERVICE.md`.
- Revisión legal final de copy y emails de contacto antes de release público.
